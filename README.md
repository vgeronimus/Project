# Simon Task Experiment (PsychoPy)

This repository contains a Simon Task experiment implemented in Python using PsychoPy.

## Files

- `Simon Task_fin.py` – final version of the experiment  
- `stimuli.csv` – stimulus definitions  
- `participants_data/` – example result files  

## Requirements

- Python 3
- PsychoPy

## How to run

```bash
python "Simon Task_fin.py"
